
# AI Chat Assistant using OpenRouter and nextJs
## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```
The application is an AI assistant made using OpenRouter API and nextJs. Using the Llama model cause its free!
